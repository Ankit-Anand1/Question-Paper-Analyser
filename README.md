Question Paper Analyser

🚀 Overview

A full-stack web application that analyzes question papers and provides insights such as trends, important topics, and predictions.

🛠️ Tech Stack

- Frontend: React.js / Next.js
- Backend: Node.js, Express.js
- Database: MongoDB
- AI Integration: OpenRouter API

✨ Features

- Upload and analyze question papers
- Trend detection and insights
- AI-based predictions
- Clean dashboard UI

⚙️ Setup

git clone <your-repo-link>
cd project
npm install
npm run dev

🔐 Environment Variables

Create a ".env" file:

API_KEY=your_key_here

📌 Note

".env" is ignored for security reasons.
